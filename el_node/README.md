# goosd_admin Backend

Node.js + Express backend scaffold.

## Scripts

```bash
npm run dev
npm start
```

## Environment

Copy `.env.example` to `.env` and adjust values as needed.

## Endpoints

- `GET /api/health`
- `POST /api/auth/login`

Default login:

```json
{
  "username": "admin",
  "password": "admin123"
}
```
